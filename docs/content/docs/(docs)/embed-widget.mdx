Rybbit supports two iframe embed options:

- [Widget Embeds](/docs/embeds/widget) show a compact live visitor widget on any web page.
- [Dashboard Embed](/docs/embeds/dashboard) shows your analytics dashboard in an iframe.

Both embeds are self-contained iframes. There is no script tag, no JavaScript on your site, and no dependency on the Rybbit tracking script.

## Which Embed Should I Use?

Use [Widget Embeds](/docs/embeds/widget) when you want a small live visitor count, chart, or top-countries summary. Widget Embeds do not make your dashboard public.

Use [Dashboard Embed](/docs/embeds/dashboard) when you want to publish the full read-only analytics dashboard. Dashboard Embed uses a generated private link, so Public Analytics is not required.
